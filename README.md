# Interface-MongoDB

###################Prerequis###################
Installer PHP 7.4
Apache2

-----------------------------------------
-------------MongoDB---------------------
$ git clone https://github.com/mongodb/mongo-php-driver.git
$ cd mongo-php-driver
$ git submodule update --init
$ phpize
$ ./configure
$ make all
$ sudo make install

-----------------------------------------------
----------------depôt git-------------------
Clônez le projet dans 
- /var/wwww/html 

------------------------------------------------
----------------Depuis le navigateur---------------
-localhost/Interface-MongoDB
